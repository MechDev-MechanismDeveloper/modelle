# MechDev Kinematics and Mechanism Models

## About MechDev
Mechanism Developer (MechDev) is a kinematics software for the design and the analysis of planar linkages, geared linkages and cam mechanisms.
More Information [https://www.mechdev.de](https://www.mechdev.de/index.php/en/)


## Kinematics and Mechanism Models

This repository contains the MechDev Files (PMD format) for the mechanisms and kinematics featured in the blog posts at
